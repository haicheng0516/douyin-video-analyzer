# 抖音视频分析

## 功能

自动操作抖音 Mac 客户端，粘贴视频链接并分析内容。

## 使用步骤

### 1. 打开抖音客户端

```bash
open -a "抖音"
```

等待 App 启动完成。

### 2. 导航到搜索页面

在抖音客户端中找到搜索入口，点击进入搜索页面。

### 3. 粘贴视频链接

- 如果链接是短链（如 `https://v.douyin.com/xxx`），先在浏览器中打开获取完整链接
- 或者直接在抖音搜索框中粘贴短链，系统会自动跳转

### 4. 获取视频信息

视频加载后，通过浏览器截图或页面解析获取：
- 视频标题
- 视频描述/文案
- 评论区内容
- 作者信息
- 章节要点（如果有）

### 5. 分析内容

将获取到的信息整理成结构化内容，包括：
- 视频主题
- 主要内容摘要
- 关键要点
- 评论区观点（可选）

## 注意事项

- 抖音客户端需要登录才能查看完整内容
- 部分视频可能需要 VIP 才能观看
- 评论区需要登录才能查看
