# 抖音视频分析

## 触发条件

用户发送抖音视频链接（douyin.com 或 v.douyin.com 开头）时自动触发。

## 工作流程

### 1. 打开抖音网页版

使用 browser 工具 navigate 到抖音视频链接：
- 如：https://v.douyin.com/xxxxx

### 2. 静音 + 倍速播放（最快速度）

进入视频页面后：
- 设置 **2倍速** 播放（最快可承受速度）
- 确保 **静音** 状态（避免打扰）

### 3. 获取视频信息

通过页面快照提取：
- 视频标题
- 视频描述/文案
- 章节要点（如果）
- 作者信息
- 评论区热门评论

### 4. 快速总结

基于视频内容快速总结：
- 视频主题
- 核心要点（1-3条）
- 关键信息

### 5. 关闭浏览器（重要！）

分析完成后，必须关闭浏览器标签页：
- 使用 browser 工具的 close 操作关闭当前标签页

## 优化策略

- **静音播放**：避免声音打扰
- **2倍速**：快速浏览内容
- **快速总结**：提炼关键信息，不照搬全文

## 注意事项

- 抖音网页版需要登录才能观看高清视频和评论
- 如果遇到登录提示，告知用户需要手动登录
- **必须在分析完成后关闭浏览器，这是最重要的步骤**
