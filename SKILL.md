# 抖音视频分析

## 触发条件

用户发送抖音视频链接（douyin.com 或 v.douyin.com 开头）时自动触发。

## 工作流程

### 1. 打开抖音客户端

```bash
open -a "抖音"
```

等待 App 启动完成（~3秒）。

### 2. 粘贴链接搜索

在浏览器中打开抖音视频链接：
- 使用 browser 工具 navigate 到视频链接
- 如：https://v.douyin.com/xxxxx

### 3. 获取视频信息

通过页面快照提取：
- 视频标题
- 视频描述/文案
- 章节要点（如果）
- 作者信息
- 评论区热门评论

### 4. 分析并汇报

将获取的信息整理成结构化内容返回给用户：
- 视频主题
- 主要内容摘要
- 关键要点

## 注意事项

- 抖音网页版需要登录才能观看高清视频和评论
- 如果遇到登录提示，告知用户需要手动登录
- 视频内容获取失败时，尝试使用网页版搜索视频标题
