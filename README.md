# douyin-video-analyzer

抖音视频分析 Skill for OpenClaw

## 功能特点

- 🔗 自动识别抖音链接并打开视频
- 🔇 静音播放 - 避免声音打扰
- ⚡ 2倍速快速浏览 - 最快速度获取内容
- 📝 智能总结 - 提炼关键信息，不照搬全文
- ❌ 自动关闭浏览器 - 分析完成后释放资源

## 触发条件

用户发送抖音视频链接（douyin.com 或 v.douyin.com 开头）时自动触发

## 使用示例

```
用户发送: https://v.douyin.com/xxxxx
→ 自动打开视频 → 静音2倍速播放 → 提取关键信息 → 总结汇报 → 关闭浏览器
```

## 输出内容

- 视频标题
- 视频描述/文案
- 核心要点（1-3条）
- 作者信息

## 注意事项

- 抖音网页版需要登录才能观看高清视频和评论
- 如果遇到登录提示，告知用户需要手动登录
